2. **AOTriton experimental hidden behind env var** — `TORCH_ROCM_AOTRITON_ENABLE_EXPERIMENTAL=1` gives 19x SDPA speedup but is not documented or default.
3. **`PYTORCH_HIP_ALLOC_CONF=backend:malloc` crashes PyTorch** — this is set by default in some ROCm shell profiles. Must be unset.

### Stability
4. **batch 2^14 (16K tokens) always NaN** — at device_batch=8, training diverges within 15 steps. Possible bf16 precision issue in small-batch gradient accumulation.

6. **Depth 10+ unreliable** — NaN at step 23 with depth 16, crash with depth 10 on second round. May be related to bf16 accumulation in deep networks.

### Performance

8. **HSA_OVERRIDE_GFX_VERSION no longer needed** — TheROCk nightlies ship native gfx1151. Document this for consumer GPU users.
